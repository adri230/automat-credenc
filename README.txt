Este script genera un archivo de texto (.docx) con nombres, DNI, recintos y fechas de un excel (.xlsx). Hay que pasarle una plantilla que se pide por consola, al igual que el excel. 

Antes de ejecutar el script hay que descargar python desde su página oficial y usar librerias.ps1 o instalar las librerias necesarias de forma manual.

Para ejecutar el script librerias.ps1 tenemos que abrir una consola de powershell como administrador y poner copiar esta línea:

	Set-ExecutionPolicy RemoteSigned

Una vez ejecutado el script librerias.ps1, se abre de nuevo una consola powershell de administrador y copiamos esta linea:

	Set-ExecutionPolicy Restricted

Enlace de la página de python para descargarlo en Windows: 
https://www.python.org/downloads/windows/
